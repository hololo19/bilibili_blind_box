# 🎯 快速验证指南

## 测试Web更新功能

### 本地测试

1. **启动Web服务器**
```bash
cd /d/ntnt
python web_server.py
```

2. **打开浏览器**
访问：`http://localhost:5000`

3. **检查功能**
- [ ] 页面正常显示
- [ ] 配置区域可见（房间号、Cookie输入框）
- [ ] 统计卡片正确显示（盈利/亏损盲盒）
- [ ] 能填写配置并保存
- [ ] 能启动/停止监听器

### Docker测试

1. **构建并启动**
```bash
cd /d/ntnt
docker-compose up -d --build
```

2. **查看日志**
```bash
docker-compose logs -f
```

3. **访问网页**
`http://localhost:5000`

## 🐛 常见问题

### Q: 网页打不开？
**A**:
```bash
# 检查端口是否被占用
netstat -ano | findstr :5000

# 或查看Docker日志
docker-compose logs -f
```

### Q: 监听器无法启动？
**A**:
1. 检查Cookie是否正确
2. 检查房间号是否正确
3. 查看监听器日志：
```bash
docker-compose logs -f blind-box-web
```

### Q: 数据没有保存？
**A**:
1. 检查 `data/` 目录是否存在
2. 检查目录权限
3. 查看服务器日志
